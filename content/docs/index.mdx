OpenProse is a programming language for AI sessions. Programs are Markdown files that agents execute as contracts.

It is meant for the work that does not fit a single prompt and does not deserve a custom framework: research pipelines, code-review choreography, content production, multi-agent orchestration with retries and fan-out. You declare what each service requires and ensures, and the runtime wires them by reading the contract. The same program runs on Claude Code, Codex, and other Prose Complete hosts. Every run leaves an inspectable trace on disk.
