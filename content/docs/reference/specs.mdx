# Canonical specs

If this site and the skill disagree, trust the skill.

The docs site is orientation. These files define how OpenProse agents should behave.

## Skill

- [`open-prose` skill](https://github.com/openprose/prose/blob/main/skills/open-prose/SKILL.md)

Agents should start here. The skill contains the activation rules, path table, command routing, host primitive adapter, and fit check.
